# Neon Maze

A mobile-friendly, responsive neon-themed maze game built with vanilla HTML, CSS, and JavaScript.

## Features
- **Procedurally Generated Mazes**: Every game provides a unique, randomly-generated perfect maze using a recursive backtracker algorithm.
- **Multiple Difficulties**: Choose between Easy, Medium, and Hard, with progressive difficulty as you complete levels.
- **Responsive Controls**: Use Arrow Keys or WASD on desktop, or swipe gestures on mobile devices to navigate.
- **Premium Aesthetics**: A sleek dark mode UI with cyber-punk style neon glowing accents to make the visuals pop.

## How to Play
1. Open up `index.html` in your web browser.
2. Select your difficulty and click **Start Game**.
3. Use your arrow keys or swipe (on touch screens) to navigate the pink player dot through the cyan glowing walls.
4. Reach the pulsing purple dot at the end to complete the level. 

## Deployment Setup
This project uses zero dependencies. To host it, simply push this repository to GitHub and enable **GitHub Pages** from the root branch. No build steps are required.
